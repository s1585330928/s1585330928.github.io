Dimensional Circle「维度回廊」：游戏内进入合作副本（如 幽影深渊）的匹配与组队入口系统。
Specter's Abyss《王座与自由》维度回廊中的 Tier 0 六人合作副本之一，英文常用名 Specter's Abyss；中文社区常称「幽影深渊」。详见本站 攻略页。
Peace / Conflict常见于世界区域或活动规则：和平期与冲突期行为约束不同（PvP 开关、争夺逻辑等），以游戏内图例为准。
Archboss大型野外 / 公会向 Boss，通常有固定参与窗口与高价值掉落，适合单独做时间表页。
Boonstone / Riftstone野外据点类目标，多与公会争夺、资源与赛季节奏相关；攻略侧重「流程、分队、时间窗」。
构筑（Build）双持武器组合 + 技能与被动取舍 + 场景定位（PVE / PVP / 坦克 / 治疗等）。本站 构筑库 为速查入口。
合约 / 日常每日或每周刷新任务线，用于金币、材料与代币的稳定来源；与 每日待办 联动勾选。
机制 / 灭团点副本 Boss 中易导致团灭的技能或流程节点；速查页应用列表或时间轴呈现。
localStorage浏览器本地存储：本站待办勾选与构筑收藏仅存于你的设备，清除站点数据会丢失；可用 今日指挥台 导出备份。
